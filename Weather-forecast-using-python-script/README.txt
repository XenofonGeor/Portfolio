In this project, visual crossing weather API is used to get weather data from 3 different locations for a 7-day forecast. Then, weather data is loaded into a MySQL database using python.

Instructions for running the application and installing any dependencies
  1) Connect the database to localhost.
  2) Install python3.8.
  3) pip install mysql-connector-python.
  4) In order to run the example sql queries, remove 'ONLY_FULL_GROUP_BY' sql mode from mySQL.
     In case MySQLWorkbench is used. Go to preferences under Modelling section > MySQL.
